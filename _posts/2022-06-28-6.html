class Solution:
    def closeStrings(self, word1: str, word2: str) -> bool:
        s1=set(word1)
        s2=set(word2)
        if s1==s2:        
            return sorted([word1.count(i) for i in s1])==sorted([word2.count(i) for i in s2])
        return False

Solution은 TC가 O(n)인 count를 최대 26번 썻고, TC가 O(26 log26) 인 sort를 썻으므로 전체의 TC가 26*C1*O(n)이다.
